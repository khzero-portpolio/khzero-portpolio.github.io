로그인 페이지

기능소개

  • 일반 로그인

  • SNS로그인 (Oauth2.0 로그인)

    • https://github.com/chicpro/gnuboard5/blob/oauth/plugin/oauth/ 을 참조하여 작업하였습니다.

    • 구글 로그인

    • 카카오 로그인

    • 라인 로그인

제품

기능소개

  • 등록한 제품 이미지 썸네일 슬라이더 처리
    • flexslider 라이브러리 사용
  • 수량 및 옵션 선택시 금액 계산 처리
    • 다국어 페이지의 경우 USD 달러로 표시 됩니다.
  • 장바구니 또는 바로구매
    • 장바구니 버튼 선택시 선택한 제품이 장바구니에 담겨지고 장바구니 페이지로 이동합니다.
    • 바로구매 버튼 선택시 선택한 제품이 장바구니에 담겨지고 주문페이지로 이동합니다.

장바구니

  • 금액처리

    • 체크 박스 체크시 수량에 따른 금액처리 스크립트 작업
    • 메인 제품이 아닌 악세서리 제품(옵션 제품)만 선택시 해외배송의 경우 배송비 추가 스크립트 처리 작업

  • 수량변경 처리

  • 장바구니 제품 삭제 기능
