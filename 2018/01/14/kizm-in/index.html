기능 소개

  • 사내 인트라넷 페이지
    • 회원 기능
      • 미로그인시 로그인 페이지로 이동
      • 회원별 썸네일 프로필 이미지 처리
      • 동시 접속 차단
        • 다른 세션에서 로그인 할 경우 현재 세션 해제
      • 로그인 기록 및 통계
        • 기간별 조회
        • 월별 통계
        • 년도별 통계
      • 현재 접속자 리스트 기능
      • 7일 이상 미접속 계정 휴면 계정처리(crontab 사용)
    • 일정 기능
      • fullCalendar.js
      • 시작일, 종료일 입력한 데이터 가공 처리
      • Ajax 처리
        • 기본 현재 월 일정 표시
        • 일자 선택시 해당 일자 일정 표시
